## Czym jest oprogramowanie QMK? :id=what-is-qmk-firmware

QMK (*Quantum Mechanical Keyboard*) to społeczność open source, skupiona na rozwijaniu urządzeń do wprowadzania danych komputerowych.
Społeczność obejmuje wszelkiego rodzaju urządzenia do wprowadzania danych, takie jak klawiatury, myszy i urządzenia MIDI.
Grupa głównych współpracowników utrzymuje [Oprogramowanie QMK](https://github.com/qmk/qmk_firmware), [Konfigurator QMK](https://config.qmk.fm),
[Narzędzia QMK](https://github.com/qmk/qmk_toolbox), [qmk.fm](https://qmk.fm) oraz tę dokumentację przy wsparciu członków społeczności, takich jak Ty.

## Dostosuj pod siebie :id=make-it-yours

QMK oferuje wiele funkcji do odkrycia oraz dużo dokumentacji referencyjnej do przejrzenia. Większość funkcji można wykorzystać, modyfikując swoją mapę klawiszy
zwaną [keymap](pl/keymap.md) oraz zmieniając kody klawiszy zwane [keycodes](pl/keycodes.md).
